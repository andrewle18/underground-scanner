# Underground Scanner
Máy quét tín hiệu trading theo hệ thống EMA 34/89 đa khung H4+H1.

**Live:** https://andrewle18.github.io/underground-scanner/

## Tính năng
- Quét đa khung H4 + H1 với EMA 34/89
- Phát hiện vùng giá trị, phân kỳ/hội tụ MACD
- Chart nến H4 + Fibonacci Extension theo Dow
- Kế hoạch lệnh tự tính (Entry/SL/TP/Volume theo quy tắc 2%)
- Tự quét theo chu kỳ + chuông báo

Dữ liệu từ Binance public API. Chạy 100% trên trình duyệt, không cần server.
